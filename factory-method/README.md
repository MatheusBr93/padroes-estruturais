# Factory Method Project

Projeto simples em Java para demonstrar o padrão Factory Method.

## Estrutura do padrão
- Product: `Notification`
- Concrete Products: `EmailNotification`, `SmsNotification`
- Creator: `NotificationCreator`
- Concrete Creators: `EmailNotificationCreator`, `SmsNotificationCreator`

## Como executar
Compile os arquivos da pasta `src` e execute `app.Main`.
